# Viewing Query Statistics !!! info "Documentation in progress" For a summary of the metrics shown in this dashboard, see [Query Statistics in the Overall Statistics dashboard](viewing-overall-statistics.md#query-statistics). ![Query statistics dashboard]({{base_path}}/images/streaming-integrator-grafana-dashboard/query_statistics_dashboard.png) This dashboard displays the following information for your current WSO2 Integrator: SI deployment: - The throughput of each query - The total query count - The memory consumed by the queries over time. - The latency of the queries over time.
